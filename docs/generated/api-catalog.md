## Totals

- Namespaces: **128**

- Rule: Custom UIs call the Host SDK first; raw RPC is for extending the SDK or backend integration.

Per-method detail (schemas, permissions, safety hints, recommended
Host SDK wrappers) lives in `api-catalog.json`; look methods up via
the MCP tool `get_api_method`.

## Raw Escape Hatches (advanced)

| Surface | Path | Note |
|---|---|---|
| `host.api` | `frontend/src/shell/template-host/api.ts` | Generic RPC proxy over every namespace. Prefer the curated host domains. |
| `call` | `frontend/src/shell/template-host/api.ts` | Raw single RPC call. Prefer a named host wrapper. |
| `listAll` | `frontend/src/shell/template-host/api.ts` | Raw auto-paged RPC call. Prefer a named host wrapper. |
| `useTemplateRpc` | `frontend/src/shell/template-host/rpc.ts` | Raw reactive RPC wrapper for templates. Prefer domain composables. |
| `host.devices.call` | `frontend/src/shell/template-host/devices.ts` | Relays an arbitrary RPC to a Shelly device. Device firmware surface, not FM. |
| `createHostDomain().call` | `frontend/src/shell/template-host/domain.ts` | Untyped namespace-scoped RPC call. Prefer callTyped or a named wrapper. |
