# Embedded Apps API (Node-RED + Grafana)

Generated by `backend/scripts/generate/` — do not edit by hand.
Regenerate with `cd backend && npm run generate`.

These are the upstream tools' own APIs, reached through FM's
authenticated proxy. Build the payload, POST it through the proxy; the
embedded tool then calls FM's `/rpc` (Node-RED) or the read-only `fm`
datasource (Grafana). The live `/api/docs/embedded-apps` endpoint lists
only the add-ons the running deployment has enabled.

## Node-RED

- Proxy base: `/node-red/red`
- Auth (external): FM session or Bearer PAT with node-red access; FM injects x-fm-node-red-proxy-secret upstream
- Auth (internal): Direct to node-red:1880 with header x-fm-node-red-proxy-secret

### `GET /node-red/red/flows`

List all flows (+ rev)

```bash
curl -H "Authorization: Bearer <FM_PAT_with_node-red_access>" https://<host>/node-red/red/flows
```

### `POST /node-red/red/flow`

Create one flow (tab) -> {id}

```bash
# Create one flow (tab)
curl -X POST https://<host>/node-red/red/flow \
  -H "Authorization: Bearer <FM_PAT_with_node-red_access>" -H "Content-Type: application/json" \
  -d '{
    "label": "Toggle switch",
    "nodes": [
      {"id":"inj1","type":"inject","name":"every 5s","repeat":"5","once":false,
       "props":[{"p":"payload"}],"payloadType":"date","wires":[["rpc1"]]},
      {"id":"rpc1","type":"fm-rpc","name":"toggle","server":"fmserver1",
       "operation":"entity.InvokeAction","paramsSource":"config",
       "paramsJson":"{\"id\":\"<entityId>\",\"action\":\"toggle\"}","wires":[["dbg1"]]},
      {"id":"dbg1","type":"debug","name":"result","wires":[]}
    ],
    "configs": []
  }'
# -> {"id":"a6d46e9613768438"}
# server "fmserver1" = the predefined fm-server node; else include it in configs.
```

### `PUT /node-red/red/flow/:id`

Update one flow

```bash
# Update one flow (replace its tab)
curl -X PUT https://<host>/node-red/red/flow/<flowId> \
  -H "Authorization: Bearer <FM_PAT_with_node-red_access>" -H "Content-Type: application/json" \
  -d '{"id":"<flowId>","label":"Toggle switch (10s)",
       "nodes":[ ...updated nodes... ],"configs":[]}'
```

### `DELETE /node-red/red/flow/:id`

Delete one flow

```bash
curl -X DELETE https://<host>/node-red/red/flow/<flowId> -H "Authorization: Bearer <FM_PAT_with_node-red_access>"
```

### `POST /node-red/red/flows`

Replace all flows (full deploy)

```bash
# Replace ALL flows at once (full deploy)
curl -X POST https://<host>/node-red/red/flows \
  -H "Authorization: Bearer <FM_PAT_with_node-red_access>" -H "Content-Type: application/json" \
  -H "Node-RED-Deployment-Type: full" \
  -d '[ {"id":"tab1","type":"tab","label":"Main"}, ...nodes and configs... ]'
```

## Grafana

- Proxy base: `/grafana/api`
- Auth (external): FM session with grafana access; FM injects X-WEBAUTH-USER upstream
- Auth (internal): Direct to grafana:3000 with header X-WEBAUTH-USER

### `POST /grafana/api/dashboards/db`

Create or update a dashboard -> {uid,url}

```bash
# Create OR update (same call; overwrite:true updates by uid/title)
curl -X POST https://<host>/grafana/api/dashboards/db -H "Content-Type: application/json" \
  -d '{
    "dashboard": {
      "title": "Fleet Energy",
      "schemaVersion": 39,
      "panels": [
        {"type":"timeseries","title":"Energy by device (kWh)",
         "datasource":{"type":"grafana-postgresql-datasource","uid":"<fm-ds-uid>"},
         "targets":[{"format":"time_series","rawSql":"SELECT $__timeGroupAlias(bucket,$__interval), device, SUM(sum_val)/1000.0 FROM device_em.energy_15min WHERE tag='\''total_act_energy'\'' AND $__timeFilter(bucket) GROUP BY 1,2"}]}
      ]
    },
    "overwrite": true
  }'
# -> {"uid":"...","url":"/grafana/d/.../fleet-energy","status":"success"}
# Query the curated views (device_em.energy_15min, mv__* aggregates), not raw tables.
```

### `GET /grafana/api/dashboards/uid/:uid`

Get a dashboard by uid

```bash
curl https://<host>/grafana/api/dashboards/uid/<uid>
```

### `GET /grafana/api/search`

List dashboards and folders

```bash
curl "https://<host>/grafana/api/search?type=dash-db"
```

### `DELETE /grafana/api/dashboards/uid/:uid`

Delete a dashboard

```bash
curl -X DELETE https://<host>/grafana/api/dashboards/uid/<uid>
```

### `GET /grafana/api/datasources/name/fm`

The provisioned read-only fm datasource

```bash
curl https://<host>/grafana/api/datasources/name/fm
```
